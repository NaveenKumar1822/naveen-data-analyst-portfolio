## 🚚 Nassau Candy Logistics Intelligence

**Domain:** Supply Chain Analytics

**Tech Stack:** Python, Pandas, Streamlit, Plotly, Apache Parquet

A production-grade analytics pipeline designed to diagnose delivery failures across a multi-factory confectionery supply chain. The project processed over 10,000 shipment records across 196 routes and restored operational accuracy by correcting severe date anomalies.

### Key Highlights
- Reverse-engineered and corrected date offsets up to 1,634 days.
- Built route-level performance intelligence.
- Developed interactive Streamlit dashboards.
- Computed Route Efficiency Scores (RES).
- Enabled SLA monitoring and operational visibility.
